# AccelMars

**AI-era company OS. One founder + AI = full team.**

AccelMars is a structured operating system for human-AI collaboration. A single human sets direction; AI agents execute at scale through contracts, skills, and an externalized knowledge base. Every decision, pattern, and process is captured in a persistent filesystem that gives stateless AI agents the memory they need to work effectively.

## What We Build

| Product | What It Does | Status |
|---------|-------------|--------|
